# CVPR Year Patterns

CVF Open Access typically exposes each CVPR year under:

```text
https://openaccess.thecvf.com/CVPR{year}
```

The all-papers view is usually:

```text
https://openaccess.thecvf.com/CVPR{year}?day=all
```

Day pages should be discovered from the year home page instead of hardcoded. This protects the skill from year-to-year date changes.

The first version treats these pages as CVPR main conference sources. Workshop collection is explicitly out of scope.
