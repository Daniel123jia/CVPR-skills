# Principles

- 只做 CVPR 论文精读，面向单篇或少量 CVPR main conference papers。
- 不扩展到 ICCV、ECCV、NeurIPS、ICML、ACL、AAAI、IJCAI 或其他会议。
- 不承担整届 CVPR 论文采集、导出、完整性检查或会议级趋势分析；这些属于 `conference-cvpr`。
- 优先基于用户提供的论文文本、本地 PDF 解析文本或明确给出的论文内容。
- 可以读取 `conference-cvpr` 产出的 `title`、`authors`、`abstract`、`pdf_url`、`paper_page_url` 作为入口。
- 如果只有 title，不做细节分析，只能给出标题级定位和证据不足提醒。
- 如果只有 title + abstract，只能做 preliminary summary，不能声称读过全文。
- 如果有全文文本，才可以做方法、实验、局限性和灵感分析。
- 输出必须落到 Markdown 文件中，供用户直接保存、复用或继续编辑。
- 缺证据时写明缺口，不用猜测填补。
