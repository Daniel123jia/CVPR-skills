# Expected Behavior

- `experiments.md` must include `Numeric Extraction Confidence`.
- For each table or figure, mark confidence as high, medium, low, or unavailable.
- Low-confidence tables must not support strong conclusions.
- Medium-confidence tables require cautious wording and manual verification.
- Every experimental number must include an evidence anchor.
- If different metrics trade off, do not write "uniformly best".
- If PDF extraction compressed the table, state that explicitly.
