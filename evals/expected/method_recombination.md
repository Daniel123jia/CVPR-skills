# Expected Route: Method Recombination

- Skill: `cvpr-idea-miner`
- Workflow: `method-recombination`
- Evidence level: `reader_notes` or `fulltext_notes`
- Output file: `outputs/computer_vision/cvpr/ideas/{year}/method_recombination.md`

Expected behavior:

- Build a method module matrix from provided notes.
- Every recombination candidate includes `evidence source`, related papers / paper_ids, proposed combination, expected benefit, risk, and evidence level.
- State that the 组合想法 is an agent proposal or user hypothesis unless the source explicitly says the same combination already exists.
- Do not present recombinations as existing paper conclusions.
- Do not invent network structures, formulas, datasets, baselines, or experimental results.
